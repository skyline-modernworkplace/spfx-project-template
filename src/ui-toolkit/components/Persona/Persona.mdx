# Persona

```tsx
import { Persona } from "ui-toolkit";
```

Displays a user with a photo and title, can be configured to show subtitle and information
as well as a call to action button.

```tsx live=true
<Persona
  photo="https://images.pexels.com/photos/210019/pexels-photo-210019.jpeg?auto=compress&cs=tinysrgb&dpr=2&h=750&w=1260"
  title="Will Spiering"
  subTitle="Software Engineer"
  info="Modern Workplace"
  callToAction="Learn More"
  linkUrl="https://wikipedia.com"
  //   orientation="vertical"
  as={Card}
/>
```

## Basic Usage

At minimum you should pass a `photo`, and `title`.

```tsx live=true bordered=true
<Persona
  photo="https://images.pexels.com/photos/210019/pexels-photo-210019.jpeg?auto=compress&cs=tinysrgb&dpr=2&h=750&w=1260"
  title="Will Spiering"
  subTitle="Software Engineer"
  info="Modern Workplace"
/>
```

## Link Url and Call To action

You can pass a `linkUrl` and it will send the user there on click. If you pass a `callToAction`, a button will be rendered for the link instead of having the entire persona be clickable.

```tsx live=true bordered=true
<div>
  <Persona
    photo="https://images.pexels.com/photos/210019/pexels-photo-210019.jpeg?auto=compress&cs=tinysrgb&dpr=2&h=750&w=1260"
    title="Will Spiering"
    subTitle="Software Engineer"
    info="Modern Workplace"
    linkUrl="https://wikipedia.com"
  />
  <Persona
    photo="https://images.pexels.com/photos/210019/pexels-photo-210019.jpeg?auto=compress&cs=tinysrgb&dpr=2&h=750&w=1260"
    title="Will Spiering"
    subTitle="Software Engineer"
    info="Modern Workplace"
    linkUrl="https://wikipedia.com"
    callToAction="Learn More"
  />
</div>
```

## Persona Card

To render the persona as a Card, use the 'as' prop and pass the 'Card' component

```tsx live=true
<Persona
  photo="https://images.pexels.com/photos/210019/pexels-photo-210019.jpeg?auto=compress&cs=tinysrgb&dpr=2&h=750&w=1260"
  title="Will Spiering"
  subTitle="Software Engineer"
  info="Modern Workplace"
  callToAction="Learn More"
  linkUrl="https://wikipedia.com"
  as={Card}
/>
```
